## Setup

- clone install `~/Development`
- Install GPGSuite (see https://github.com/nicholasjhenry/cheat/blob/master/gpg)
- Add key
- Install rc file manager (https://github.com/thoughtbot/rcm)
- run `rcup ~/Development/dotfiles`
- Install `./fonts`
